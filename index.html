How I present each lab (suggested contents)

  • Overview: What the lab demonstrates and real-world use-case.
  • Topology Image: PNG/SVG screenshot of the topology.
  • Files to download: startup-configs, .unl/.zip, VM export, scripts, and test pcap if applicable.
  • Step-by-step Guide: Exact commands and GUI steps to build the lab from scratch.
  • Verification: Commands and expected outputs (ping, OMP routes, ISE logs).
  • Notes & Troubleshooting: Known pitfalls and quick fixes.

How to add new lab pages and files

  1. Create a folder under labs/ (e.g. labs/sdwan/).
  2. Add an index.html inside that folder with detailed steps and screenshots.
  3. Place topology images and downloadable files in the same folder (example: sdwan-topology.png, configs.zip).
  4. Link the detail page from the main portfolio card (like the "View Details" button above).
  5. Commit and push to GitHub — your Pages site will serve these files automatically.
